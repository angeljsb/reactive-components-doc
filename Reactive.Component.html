Class: Component

Reactive.Component(template, initialState)

new Component(template, initialState)

Clase de la que heredan todos los componentes reactivos y agrupa la funcionalidad común entre ellos. Los métodos y propiedades de esta clase son las que se podrán llamar para todos los componentes reactivos y no deben ser sobreescritas en estos
Parameters:
Name Type Description
template function Función que devuelve un elemento html condicionado por el estado y props actuales, el cual será renderizado cada vez que estos cambien
initialState any El estado inicial del componente reactivo
